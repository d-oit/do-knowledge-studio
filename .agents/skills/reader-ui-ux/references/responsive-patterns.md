# Responsive Layout Patterns

Responsive design patterns for reader/admin interfaces.

## Viewport Breakpoints
- Mobile: < 640px (bottom tab bar, stacked layout)
- Tablet: 640-1024px (horizontal nav, 2-column max)
- Desktop: > 1024px (persistent sidebar, full layout)

## Touch Targets
- Minimum 44x44px on mobile
- Adequate spacing between interactive elements

## Progressive Enhancement
- Core content accessible without JS
- Enhanced interactions with JS enabled
