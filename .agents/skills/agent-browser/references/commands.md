# Agent Browser Command Reference

Complete command reference for agent-browser CLI.

## Core Commands

| Command | Description |
|---------|-------------|
| `open <url>` | Navigate to URL |
| `snapshot` | Get page content |
| `snapshot -i` | Get interactive elements with refs |
| `click <ref>` | Click element |
| `fill <ref> <text>` | Fill input field |
| `screenshot` | Take screenshot |
| `screenshot --annotate` | Screenshot with numbered labels |
| `wait <ms>` | Wait milliseconds |
| `batch "cmd1" "cmd2"` | Run multiple commands |
| `close` | Close session |
